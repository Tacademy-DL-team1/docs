## 🗣️ 주요 논의 내용 (강사님 피드백)

### 1. 프로젝트 접근 방향

- 복잡한 문제부터 해결하려 하지 말고, 현재 구현할 수 있는 단순한 문제부터 단계적으로 진행해야 한다.

- 사진의 구성이나 구도가 복잡한 이미지보다 정면 사진처럼 객체를 탐지하기 쉬운 이미지부터 모델 성능을 확인해야 한다.

- 정면 사진에서 객체를 정상적으로 탐지한다면 이후 복잡한 구도의 이미지로 확장하고, 정면 사진에서도 탐지하지 못한다면 해당 단계에서 발생한 문제를 먼저 해결해야 한다.

- 객체 크기가 크고 해상도가 높은 이미지부터 학습과 검증에 활용하는 것이 필요하다.

- 구글과 Roboflow에서 확보한 이미지를 먼저 활용하고, 이후 AI-Hub 이미지로 확장하는 방향을 제안하였다.

- 하나의 방법만 깊게 시도하기보다 여러 방식과 모델을 다양하게 적용하면서 프로젝트 방향을 설정해야 한다.

- 하나의 모델만 사용하는 방식 외에도 여러 모델의 결과를 종합하거나 기존에 구축된 모델을 활용하는 방안을 검토할 필요가 있다.

- 다른 사람이 구축한 학습 데이터와 학습에 사용한 데이터 규모를 확인하여, 프로젝트에 필요한 데이터 수를 판단해야 한다.

### 2. `worker` 클래스 탐지 문제

- 현재 학습된 `worker` 클래스가 작업자의 전신이 아닌 신체 일부만 탐지하는 문제가 확인되었다.

- 별도로 파인튜닝하지 않은 사전학습 모델의 `person` 클래스는 작업자를 비교적 잘 탐지하므로, `worker` 클래스를 새로 학습하지 않고 기존 모델을 사용하는 방안이 제안되었다.

- 다만 `worker` 클래스가 신체 일부만 탐지하는 원인은 먼저 확인할 필요가 있다.

- 작업자의 전신이 여러 영역으로 분할되지 않고 하나의 영역으로 라벨링된 이미지만 선별하여 다시 학습한다.

- 학습 결과 작업자의 전신을 정상적으로 탐지한다면, `worker` 클래스는 전신 전체를 하나의 객체로 라벨링해야 한다고 판단할 수 있다.

- 전신 라벨링 데이터만 사용해도 신체 일부만 탐지한다면, `worker` 클래스를 별도로 학습하지 않고 사전학습 모델의 `person` 클래스를 활용한다.

- 단순히 사람을 탐지하는 것에서 나아가, `person`과 안전모 등의 객체가 함께 탐지되는 경우 작업자로 판단하는 알고리즘을 구성하는 방안을 검토한다.

### 3. `hook` 및 `lifeline` 소형 객체 탐지

- 안전고리와 생명줄은 이미지에서 차지하는 크기가 작아 기존 모델이 탐지하지 못할 가능성이 있다.

- 데이터를 추가한 후에도 탐지 성능이 개선되지 않는다면, 소형 객체 탐지에 적합한 다른 모델을 적용할 필요가 있다.

- `hook`과 `lifeline`처럼 작은 객체를 탐지할 수 있는 모델로 RT-DETR이 제안되었다.

- RT-DETR의 적용 가능성과 탐지 성능을 확인할 필요가 있다.

- 참고 문서: [Ultralytics RT-DETR 개요](https://docs.ultralytics.com/ko/models/rtdetr#%EA%B0%9C%EC%9A%94)

### 4. 생명줄 탐지 방법

- 생명줄은 직선과 유사한 형태를 가지므로 객체 탐지 모델 외에 라인 검출과 에지 검출 기법을 활용할 수 있다.

- 이미지에서 여러 직선을 탐지한 뒤, 색상이나 구조 등의 특징을 기준으로 필요한 생명줄만 선별하는 알고리즘을 구성하는 방안을 검토하였다.

## 🎯 회의 결과

- 탐지가 쉬운 정면 이미지와 크기가 크고 해상도가 높은 객체부터 학습 및 검증을 진행하기로 하였다.

- 하나의 모델이나 방법에 집중하기보다 다양한 모델과 기존 학습 데이터를 활용하여 성능을 비교하기로 하였다.

- `worker` 클래스 문제는 전신이 하나의 영역으로 라벨링된 이미지만 선별하여 다시 학습한 뒤 원인을 확인하기로 하였다.

- 재학습 후에도 신체 일부만 탐지한다면, 별도의 `worker` 클래스 대신 사전학습 모델의 `person` 클래스를 활용하기로 하였다.

- `person`과 안전모 등의 탐지 결과를 결합하여 작업자를 판별하는 알고리즘도 검토하기로 하였다.

- `hook`과 `lifeline`의 탐지 성능이 개선되지 않는 경우 RT-DETR과 같은 소형 객체 탐지 모델을 적용하기로 하였다.

- 생명줄은 라인 검출 및 에지 검출 기법을 적용한 뒤 색상과 구조를 기준으로 선별하는 방안을 검토하기로 하였다.
